# Internet Archive Uploader

This project provides a **lightweight Python script** to automatically
upload files to the **Internet Archive** using the official
internetarchive library.
It is designed to be simple, reliable, and easy to integrate into your
personal workflows.

# Requirements
-   Python 3.8+
-   Required Python packages:
    -   internetarchive
    -   python-dotenv

# Setup

### Create a `.env` file
    access-key=YOUR_ACCESS_KEY
    secret-key=YOUR_SECRET_KEY
    username-username=IDENTIFIER (WHATEVER YOU WANT)
### Run script
    python main.py
    Also there is a "run.bat" file to execute on windows
